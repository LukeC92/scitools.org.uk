    Official releases

    Official releases can be found here. These releases contain source, documentation and instructions on how to build and install Iris under Linux.

    GitHub repository

    We strongly encourage people to contribute to Iris and for this type of development we suggest you obtain the latest version of the code from our github repository. Assuming you have git installed, enter the following command into a terminal: 

    git clone https://github.com/SciTools/iris.git
