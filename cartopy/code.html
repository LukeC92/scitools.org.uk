    Cartopy is published as open-source software under an LGPLv3 license.

    Getting the code

    The Cartopy project uses git and github for managing its code.

    Once you have git installed, enter the following command to obtain the most recent code from github: 

    git clone https://github.com/SciTools/cartopy.git

    There are also official releases available from here.
